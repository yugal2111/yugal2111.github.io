Arrow-Rotation-Application

  • Tech Stack: Android Studio, XML & Java.
  • Github URL: Project Link

In this app we create one arrow rotation Application.

When the screen is vertical then the arrow should point towards the sky. When the screen is horizontal then this B1, B2,B3 and B4 button is Visible and the arrow should keep rotating in a clockwise direction.

When we Tap on B1 button arrow is stop towards the B1 button direction when we double on and long pressed on B1 button then arrow is stop at opposite its direction

When we Tap on B2 button arrow is stop towards the B2 button direction when we double on and long pressed on B2 button then arrow is stop at opposite its direction

When we Tap on B3 button arrow is stop towards the B3 button direction when we double on and long pressed on B3 button then arrow is stop at opposite its direction

When we Tap on B4 button arrow is stop towards the B4 button direction when we double on and long pressed on B4 button then arrow is stop at opposite its direction
